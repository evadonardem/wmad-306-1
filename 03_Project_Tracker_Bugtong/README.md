# Project Tracker - Rubric Mapping & Usage Guide

This is a full-stack Project Tracker built with Laravel 12, React 18, Material UI 7, Inertia.js, and Docker Compose. It implements all rubric requirements for WMAD 306, including:

## How to Run

### Prerequisites
- Docker & Docker Compose
- Node.js 18+, npm

### Setup
1. **Clone repo**
2. `cd 03_Project_Tracker_Bugtong`
3. `docker compose -f compose.yaml up --build`
4. In another terminal: `docker compose -f compose.yaml exec web npm run build`
5. Visit [http://localhost:8000](http://localhost:8000)

### Database
- MySQL runs in Docker, seeded with demo data
- To reset: `docker compose -f compose.yaml exec web php artisan migrate:fresh --seed`

### Development
- Frontend: edit files in `resources/js/`
- Backend: edit files in `app/`, `routes/`, `database/`
- Hot reload: `npm run dev` (inside web container)
